O Guizap é um chat básico que simula a troca de mensagens em tempo real entre usuários conectados. O projeto utiliza o sistema de pub/sub do Flet, permitindo que mensagens sejam distribuídas instantaneamente para todos os participantes.

Funcionalidades:
Envio de mensagens em tempo real
Entrada no chat com nome de usuário
Comunicação entre múltiplos usuários
Interface gráfica simples e interativa
Atualização automática das mensagens

Tecnologias utilizadas:
Python 3
Flet
